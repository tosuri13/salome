# プロダクト概要

Salomeは、Discord上で動作するAIアシスタント「Claudeお嬢様」のリファクタリング版です。前身プロジェクト「Lady Claude」をモダンな技術スタックで再構築しています。

## コア機能

- **Discord Slash Commands**: ユーザーからのコマンドを受け付けるインタラクティブなBot
- **AIアシスタント**: Amazon Bedrock (Claude) を活用した「お嬢様」キャラクターでの応答生成
- **定期実行サービス**: ゴミ出しリマインドなど、スケジュールベースの通知機能
- **サーバーレス処理**: Lambda + SNSによるコマンドごとの非同期処理振り分け

## ターゲットユースケース

| コマンド | 用途 |
|---------|------|
| `ask` | Claudeお嬢様への自由な質問 |
| `minecraft` | Minecraft Serverの管理・操作 |
| `recipe` | 料理レシピの提案 |
| `garbage` | ゴミ出し日のリマインド通知 |

## バリュープロポジション

- お嬢様キャラクターによる親しみやすいインタラクション
- サーバーレスアーキテクチャによる低コスト運用
- Slash Commandsによる直感的なユーザー体験

## リファクタリング目標

前身プロジェクト「Lady Claude」からの改善点：
- AWS SAM → **AWS CDK** によるインフラ管理の効率化
- 独自AI実装 → **Strands Agents + LiteLLM** によるモダンなエージェント構成
- モノレポ化による機能・モジュールの整理

## 開発方針

- **個人プロジェクト**: 大規模なCI/CDやテストは不要
- **実機テスト**: 本番環境での動作確認を重視
- **漸進的改善**: 必要に応じて構成を改善
